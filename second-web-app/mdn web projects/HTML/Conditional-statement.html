Conditional Statements

Conditional Statements are Used if to specify a block of code to be executed, if a specified condition is true. Use else to specify a block of code to be executed, if the same condition is false.

Type of Conditional Statements:

There are 5 types of Conditional Statements :
  1. if
  2. if-else
  3. else if
  4. nested-if
  5. swichcase

1. IF statements

in IF statement if the condition is false, another block of code can be executed.
